### 🏅 Atlantic Metals — Production E-Commerce Platform

<div align="center">

[![Live Demo](https://img.shields.io/badge/🌐%20Live%20Demo-atlantic--metals.onrender.com-00E5FF?style=for-the-badge)](https://atlantic-metals.onrender.com)
[![GitHub](https://img.shields.io/badge/GitHub-Source%20Code-181717?style=for-the-badge&logo=github)](https://github.com/hcmadala/atlantic-metals)

</div>

> **Canada's Precious Metals Dealer** — A full-stack e-commerce platform for Gold, Silver, Platinum, and Palladium. Built from the ground up with production engineering standards — not a tutorial clone.

**📐 System Architecture**
```
Client → Express Router → JWT Middleware → Input Validation → Handler → MySQL (pooled)
```

| Metric | Detail |
|--------|--------|
| 🔗 **API Surface** | 12+ RESTful endpoints — auth, products, cart, orders |
| ⚡ **Query Performance** | ~35% latency reduction via indexed FK columns + connection pooling |
| 🔐 **Authentication** | Stateless JWT + `httpOnly` cookies — eliminates XSS token exposure |
| 🗄️ **Database** | 4-table MySQL schema (3NF) with FK constraints and transactional rollback |
| 🛒 **Cart Integrity** | Session-linked DB rows persist across logout — zero data loss |
| 👥 **Concurrency** | Architected to support 1,000+ simultaneous sessions |
| 🛡️ **Security** | Parameterized queries throughout — SQL injection prevention by design |

**🔧 Key Engineering Decisions**
- **Node.js** — non-blocking I/O model critical when cart reads and inventory writes run concurrently
- **JWT over sessions** — keeps auth stateless and horizontally scalable
- **MySQL** — ACID compliance and relational integrity across order transactions
- **Parameterized queries** — SQL injection prevention by design, not afterthought

**🏷️ Stack:** `Node.js` `Express` `MySQL` `JWT` `REST API` `HTML/CSS/JS` `Render`
